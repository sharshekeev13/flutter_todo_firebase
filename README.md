<h2 align="center"> How to run </h2>

1. Clone the repo
   ```sh
   https://github.com/sharshekeev13/flutter_todo_firebase.git
   ```
   
2. Install all the packages by typing
   ```sh
   flutter pub get
   ```
   
3. Run the App
   ```sh
   flutter run
   ```
